MERGE:    micropip.install('pypdf') — PdfWriter.append() por archivo
COMPRESS: micropip.install('pypdf') — compress_identical_objects + img.replace(quality=N)
SPLIT:    micropip.install('pypdf') — modo all: PdfWriter por pagina / modo range: parseRange

          NOTA: La conversion real de PDF a imagen requiere un renderizador de PDF
          (como pdf2image/poppler) que no esta disponible en Pyodide aun.
          La version actual crea imagenes blancas del tamano correcto como placeholder.
          Para renderizado real, evaluar: pdfjs-dist en JS puro (canvas) como alternativa.

PYTHON RECETAS — todas sin servidor
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
MERGE:    PdfWriter().append(BytesIO(bytes)) x N -> write(output)
COMPRESS: compress_identical_objects(True,True) + img.replace(quality=N)
SPLIT ALL: por pagina: PdfWriter().add_page(page).write(out) -> array
SPLIT RANGE: PdfWriter() con paginas seleccionadas -> write(out)
ROTATE:   page.rotate(90|180|270)
REORDER:  writer.add_page(reader.pages[i]) en nuevo orden
DELETE:   if i not in to_delete: writer.add_page(page)
PROTECT:  writer.encrypt(password, algorithm="AES-256")
UNLOCK:   reader.decrypt(password); writer.append(reader)
WATERMARK: fpdf2 para crear pagina watermark + merge con cada pagina
PDF TO TEXT: page.extract_text() -> Blob text/plain
PDF TO JPG: USAR PDF.js + canvas.toDataURL() — NO Pyodide para esto
