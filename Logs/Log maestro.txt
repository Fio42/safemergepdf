MERGE PDF (Pyodide + pypdf):
  PdfWriter.append(BytesIO(bytes)) por cada archivo
  writer.write(output) -> descarga merged.pdf

COMPRESS PDF (Pyodide + pypdf + Pillow — Web Worker v7.8):
  TODO corre en un Web Worker creado via Blob URL (nunca bloquea UI)
  Comunicacion via postMessage con ArrayBuffer zero-copy
  5 niveles: SCREEN(15%), EBOOK(30%), BALANCED(50%), HIGH(72%), PRINT(88%)
  Limite: 200MB. PIL re-encode de imagenes embebidas.
  compress_identical_objects + img.replace(quality=N) con try/except
  instalar: await pyodide.loadPackage(['micropip','pillow'])
            + micropip.install('pypdf')
  -> descarga filename-compressed.pdf

SPLIT PDF (Pyodide + pypdf):
  readTotalPages() via PDF.js al cargar — sin esperar Pyodide
  writer.add_page(page) por paginas del rango -> split.pdf

PDF->JPG (PDF.js, NO Pyodide):
  page.render() -> canvas -> toDataURL('image/jpeg') -> page-N.jpg

JPG->PDF (Pyodide + Pillow):
  Image.open(BytesIO(bytes)) -> img.save(buf, format='PDF')
  PdfWriter().append() -> filename.pdf

ROTATE PDF (Pyodide + pypdf + PDF.js para preview):
  pageRotations = {} — acumulador de angulos por pagina
  rotatePreview(delta): suma delta a todas las paginas target + re-render PDF.js
  Botones: Left(-90), Flip(180), Right(+90). Scope: all o range.
  doRotate(): pasa rot_map_json al Python, page.rotate(angle)
  Preview con PDF.js se carga al subir archivo (sin esperar Pyodide)
  Layout: 2 columnas (controles | preview)
  -> filename-rotated.pdf

PROTECT PDF (Pyodide + pypdf):
  writer.clone_reader_document_root(reader)
  writer.encrypt(user_password=pw, owner_password=None, use_128bit=True)
  UI: 2 campos (password + confirm), validacion coincidencia, toggle visibilidad
  -> filename-protected.pdf

UNLOCK PDF (Pyodide + pypdf):
  Password es OPCIONAL — auto-detecta si el PDF esta encriptado
  reader.decrypt(pw) — casos:
    0 = password incorrecta -> error claro
    1 = user password correcta
    2 = owner password correcta
    PDF no encriptado -> descarga igual con nota informativa
  writer.clone_reader_document_root(reader) -> writer.write(out)
  -> filename-unlocked.pdf

EDIT PDF (PDF.js + canvas API + .text-box divs + PDF builder JS manual):
  NO usa Pyodide — 100% JavaScript nativo
  Modos: select, text, draw, highlight, image, firma (modal canvas)
  Texto: .text-box div con barra azul draggable (.text-box-header con .text-box-drag-icon)
    En modo Select: annCanvas.pointerEvents='none' -> clicks pasan a .text-box debajo
    textInputs[pageNum] = array de .text-box wrapper divs
    downloadPDF(): box.querySelector('textarea') para el texto, offset BAR_H=22px
  Draw/highlight: paths {x,y}[] guardados en annotations[pageNum]
  Imagenes: img-overlay divs con drag+resize
  Firma: modal con sigCanvas -> PNG -> img-overlay
  Descarga: buildPDFFromJpegs() — renderiza paginas a JPEG 2x + ensambla PDF 1.4
    manual con XObjects. Nunca modificar buildPDFFromJpegs sin leer el LOG.
  resetAll(): container.querySelectorAll('.text-box,.img-overlay').forEach(el=>el.remove())
  Footer usa id="footerEl" — preservarlo siempre.
  -> filename-edited.pdf

DELETE PAGES (PDF.js + Pyodide + pypdf):
  PDF.js renderiza thumbnails de todas las paginas
  Click en thumbnail -> marca en rojo (Set selectedPages)
  Botones: "Select all" / "Deselect all"
  Validacion: no se puede eliminar TODAS las paginas (remaining > 0)
  Python: writer.add_page solo si (i+1) not in to_delete
  Dropzone: NUNCA onclick en el div padre — solo el boton hijo (Regla #31)
  -> filename-deleted.pdf

REORDER PAGES (PDF.js + Pyodide + pypdf — UX overhaul v8.1):
  PDF.js renderiza thumbnails (grid visual)
  Drag system: pointer events (pointerdown/pointermove/pointerup) — NO HTML5 drag API
    Ghost: clone del thumbnail, position:fixed, sigue al cursor
    Drop line: linea vertical 3px brand color, position:fixed, indica donde cae
    dropSrcIndex: indice de la pagina que se arrastra
    En pointerup: splice pageOrder[] y re-render
  Drop zone completa clickeable: onclick en div completo + stopPropagation en boton
  Python: writer.add_page(reader.pages[i]) en nuevo orden (pageOrder[])
  -> filename-reordered.pdf
